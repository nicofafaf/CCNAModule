# CCNA Modul 8-10 Quiz

Interaktives Lernquiz für die **Checkpoint-Prüfung: Kommunikation zwischen Netzwerken** (CCNA 1 v7, Module 8–10).

## Features

- **76 Prüfungsfragen** 1:1 aus der offiziellen Checkpoint-Prüfung
- **6 Lernmodi**: Lernen, Üben, Prüfung (60 Min), Fehler wiederholen, Schwächen trainieren, Nach Modul
- **Eselsbrücken** zu jeder Frage zum Auswendiglernen
- **Sofort-Feedback** mit Erklärungen
- **Spaced Repetition**: Beherrschungsgrad pro Frage (0–100%)
- **Bilder & Exhibits** für Diagramm- und Konfigurationsfragen
- **Fortschritt** wird lokal im Browser gespeichert

## Lernmodi

| Modus | Beschreibung |
|-------|-------------|
| 📖 Lernmodus | Feedback + Eselsbrücke nach jeder Antwort |
| 🎯 Übungsmodus | Alle Fragen in zufälliger Reihenfolge |
| ⏱️ Prüfungsmodus | 60 Min Timer, Ergebnis erst am Ende |
| 🔄 Fehler wiederholen | Nur falsch beantwortete Fragen |
| 💪 Schwächen trainieren | Fragen mit niedrigster Beherrschung |
| 📚 Nach Modul | Modul 8, 9 oder 10 einzeln |

## Starten

Öffne `index.html` im Browser – keine Installation nötig.

Oder mit lokalem Server:

```bash
npx serve .
```

## Module

- **Modul 8**: Netzwerkschicht (IP, IPv6, Routing, Gateway)
- **Modul 9**: Address Resolution (ARP, Switching, Ethernet)
- **Modul 10**: Cisco IOS (Router-Konfiguration, Boot, Sicherheit)

## Quelle

Fragen basieren auf [ITExamAnswers.net – CCNA 1 v7 Modules 8-10](https://itexamanswers.net/ccna-1-v7-modules-8-10-communicating-between-networks-exam-answers.html).
